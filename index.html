JavaScript Cheat Sheet

Браузерный JavaScript

  • document.getElementById(): Возвращает элемент по его ID. 
    let element = document.getElementById('myId'); console.log(element);
  • querySelector(): Возвращает первый элемент, соответствующий указанному CSS-селектору. 
    let firstDiv = document.querySelector('div'); console.log(firstDiv);
  • addEventListener(): Добавляет обработчик события на элемент. 
    element.addEventListener('click', () => { console.log('Clicked!'); });
  • setTimeout(): Запускает функцию через заданный промежуток времени. 
    setTimeout(() => { console.log('Hello after 1 second'); }, 1000);
  • setInterval(): Выполняет функцию через определенные интервалы времени. 
    let intervalId = setInterval(() => { console.log('Every second'); }, 1000);
  • fetch(): Выполняет HTTP-запрос для получения данных. 
    fetch('https://api.example.com/data')
        .then(response => response.json())
        .then(data => console.log(data));
  • localStorage: Позволяет сохранять данные в браузере без срока действия. 
    localStorage.setItem('key', 'value');
    console.log(localStorage.getItem('key')); // 'value'
  • sessionStorage: Позволяет сохранять данные в браузере, которые будут доступны только в рамках одной сессии. 
    sessionStorage.setItem('key', 'value');
    console.log(sessionStorage.getItem('key')); // 'value'
  • console.log(): Выводит информацию в консоль браузера. 
    console.log('Hello, world!');
  • alert(): Отображает диалоговое окно с сообщением. 
    alert('Hello, world!');

Работа с числами

  • NaN: "Не число", возникает, если результат операции не является числом. 
    console.log(0 / 0); // NaN
  • Infinity: Бесконечность (положительная или отрицательная). 
    console.log(1 / 0); // Infinity
  • toFixed(): Форматирует число с заданным количеством десятичных знаков. 
    console.log((2.4567).toFixed(2)); // "2.46"
  • Math: Математический объект с функциями. Math.round(): округление. 
    console.log(Math.round(2.6)); // 3
  • parseFloat(), parseInt(): Преобразуют строку в число. 
    console.log(parseInt("42px")); // 42
  • isNaN(): Проверяет, является ли значение NaN. 
    console.log(isNaN("Hello")); // true

Работа со строками

  • trim(): Убирает пробелы с обеих сторон строки. 
    console.log("  Hello  ".trim()); // "Hello"
  • split(): Разбивает строку на массив подстрок. 
    console.log("a,b,c".split(",")); // ["a", "b", "c"]
  • replace(): Заменяет часть строки на другую. 
    console.log("Hello World".replace("World", "JavaScript")); // "Hello JavaScript"
  • length: Возвращает длину строки. 
    console.log("JavaScript".length); // 10
  • toLowerCase(), toUpperCase(): Преобразуют строку в нижний или верхний регистр. 
    console.log("Hello".toUpperCase()); // "HELLO"
  • includes(): Проверяет, содержит ли строка указанное значение. 
    console.log("JavaScript".includes("Script")); // true
  • charAt(): Возвращает символ по указанному индексу. 
    console.log("Hello".charAt(1)); // "e"
  • indexOf(): Возвращает индекс первого вхождения подстроки. 
    console.log("Hello World".indexOf("World")); // 6
  • startsWith(): Проверяет, начинается ли строка с указанного значения. 
    console.log("JavaScript".startsWith("Java")); // true
  • endsWith(): Проверяет, заканчивается ли строка на указанное значение. 
    console.log("JavaScript".endsWith("Script")); // true
  • slice(): Возвращает подстроку, вырезанную по указанным индексам. 
    console.log("JavaScript".slice(0, 4)); // "Java"
  • replaceAll(): Заменяет все вхождения подстроки. 
    console.log("apple apple".replaceAll("apple", "banana")); // "banana banana"
  • padStart(): Дополняет строку указанным символом слева до нужной длины. 
    console.log("5".padStart(3, "0")); // "005"
  • padEnd(): Дополняет строку указанным символом справа до нужной длины. 
    console.log("5".padEnd(3, "0")); // "500"
  • trimStart(): Удаляет пробелы в начале строки. 
    console.log("   Hello".trimStart()); // "Hello"
  • trimEnd(): Удаляет пробелы в конце строки. 
    console.log("Hello   ".trimEnd()); // "Hello"

Работа с объектами

  • Object.keys(): Возвращает массив ключей объекта. 
    const obj = {a: 1, b: 2}; console.log(Object.keys(obj)); // ["a", "b"]
  • Object.values(): Возвращает массив значений объекта. 
    console.log(Object.values(obj)); // [1, 2]
  • Object.entries(): Возвращает массив пар [ключ, значение]. 
    console.log(Object.entries(obj)); // [["a", 1], ["b", 2]]
  • hasOwnProperty(): Проверяет, есть ли свойство у объекта. 
    console.log(obj.hasOwnProperty("a")); // true
  • JSON.stringify() — это метод в JavaScript, который преобразует объекты и массивы в строку формата JSON. Это полезно для передачи данных на сервер или сохранения их в localStorage. 
    
const user = {
    name: "Alice",
    age: 25,
    occupation: "Developer"
  };
  
  // Преобразуем объект в JSON-строку
  const jsonString = JSON.stringify(user);
  
  // Выводим результат
  console.log(jsonString);
  
  // Результат
  // {"name":"Alice","age":25,"occupation":"Developer"}
  • JSON.parse()— это метод в JavaScript, который преобразует строку формата JSON обратно в JavaScript-объект. Это полезно для обработки данных, полученных с сервера, или для извлечения данных из хранилища, таких как localStorage. 
    // JSON-строка
const jsonString = '{"name":"Alice","age":25,"occupation":"Developer"}';
                                
// Преобразуем JSON-строку в JavaScript-объект
const user = JSON.parse(jsonString);
                                
// Выводим результат
console.log(user.name);        // Alice
console.log(user.age);         // 25
console.log(user.occupation);  // Developer
  • for...in: Перебирает свойства объекта. 
    const user = {
    name: "John",
    age: 30,
    occupation: "Developer"
  };
  
  for (let key in user) {
    console.log(`${key}: ${user[key]}`);
  }
  
  // Результат:
  // name: John
  // age: 30
  // occupation: Developer

Работа с массивами

  • Array.isArray(): Проверяет, является ли значение массивом. 
    console.log(Array.isArray([1, 2, 3])); // true
  • push(): Добавляет элемент в конец массива. 
    let arr = [1, 2]; arr.push(3); console.log(arr); // [1, 2, 3]
  • pop(): Удаляет последний элемент из массива и возвращает его. 
    let last = arr.pop(); console.log(last); // 3
  • shift(): Удаляет первый элемент из массива и возвращает его. 
    let first = arr.shift(); console.log(first); // 1
  • unshift(): Добавляет элемент в начало массива. 
    arr.unshift(0); console.log(arr); // [0, 1, 2]
  • forEach(): Выполняет заданную функцию для каждого элемента массива. 
    const fruits = ["apple", "banana", "cherry", "date"];

fruits.forEach((fruit, index) => {
  console.log(`Fruit ${index + 1}: ${fruit}`);
});
                                
// Вывод:
// Fruit 1: apple
// Fruit 2: banana
// Fruit 3: cherry
// Fruit 4: date
                                
    const numbers = [1, 2, 3, 4, 5];
    let sum = 0;
    
    numbers.forEach(number => {
      sum += number;
    });
    
    console.log(`Сумма чисел: ${sum}`); 
    // Вывод: Сумма чисел: 15
  • map(): Создает новый массив, заполняя его результатами вызова указанной функции для каждого элемента. 
    const products = [
    { name: 'Laptop', price: 1000 },
    { name: 'Phone', price: 500 },
    { name: 'Tablet', price: 300 },
];

// Используем map() для создания нового массива с именами продуктов
const productNames = products.map(product => product.name);
                                
console.log(productNames); // ['Laptop', 'Phone', 'Tablet']
  • filter(): Создает новый массив с элементами, которые прошли проверку заданной функции. 
    const students = [
    { name: 'Alice', age: 22 },
    { name: 'Bob', age: 17 },
    { name: 'Charlie', age: 19 },
    { name: 'David', age: 15 },
];
                                
// Сначала отфильтруем студентов старше 18 лет
const adults = students.filter(student => student.age > 18);
                                
// Теперь получим массив имен этих студентов
const adultNames = adults.map(adult => adult.name);
                                
console.log(adultNames); // ['Alice', 'Charlie']
  • find(): Возвращает первое значение, удовлетворяющее условию, заданному в функции. 
    const users = [
    { name: "John", age: 25 },
    { name: "Alice", age: 30 },
    { name: "Bob", age: 35 }

const foundUser = users.find(user => user.age > 30);

console.log(foundUser); 
// Вывод: { name: "Bob", age: 35 }
  • some(): Проверяет, удовлетворяет ли хотя бы один элемент массива условию, заданному в функции. 
    const users = [
    { name: "John", active: false },
    { name: "Alice", active: true },
    { name: "Bob", active: false }
    ];

    const hasActiveUser = users.some(user => user.active);

    console.log(hasActiveUser); 
    // Вывод: true
  • reduce(): Применяет функцию к аккумулятору и каждому элементу массива (слева направо) для уменьшения его до одного значения. 
    let sum = arr.reduce((acc, num) => acc + num, 0); console.log(sum); // 0
  • flat():Удаляет уровни вложенности в зависимости от указанной глубины. 
    // Вложенный массив с двумя уровнями вложенности
const nestedArray = [1, 2, [3, 4, [5, 6]], 7];
                                
// Применение flat() без указания глубины (по умолчанию глубина = 1)
const flattenedArray = nestedArray.flat();
                                
console.log(flattenedArray);
// Результат: [1, 2, 3, 4, [[[[[5, 6]]]]], 7]
                                
// Применение flat() с глубиной 2
const fullyFlattenedArray = nestedArray.flat(infinity);
                                
console.log(fullyFlattenedArray);
// Результат: [1, 2, 3, 4, 5, 6, 7]

Дата/время

  • getFullYear(), getMonth(), getDate(): Получают части даты. 
    let date = new Date(); console.log(date.getFullYear());

Асинхронность

  • Promise: Объект асинхронной операции. 
    let promise = new Promise((resolve, reject) => { resolve('Done!'); });
  • async/await: Пишем асинхронный код как синхронный. 
    async function asyncFunc() { let result = await promise; console.log(result); }

Таймеры

  • setTimeout(): Выполняет функцию через определённое время. 
    setTimeout(() => console.log("Hello"), 1000);
  • setInterval(): Повторяет выполнение функции с заданным интервалом. 
    setInterval(() => console.log("Tick"), 1000);

Оператор ?

  • ?: Позволяет безопасно обращаться к вложенным свойствам. 
    let obj = { a: { b: { c: 1 } } }; console.log(obj?.a?.b?.c); // 1
